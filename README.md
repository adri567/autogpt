AutoGPT is a Xcode Source Editor extension that enhances productivity by leveraging the capabilities of GPT. It helps you to automate common coding tasks and streamline your workflow. AutoGPT uses the OpenAI's API and gives you the option to choose between four models Davinci, Curie, Babbage and Ada. AutoGPT supports also ChatGPT which uses GPT-3.5 and GPT-4.

# 🚨 Requirements

* Xcode 13.0+
* An OpenAI API Key
  * https://platform.openai.com/account/api-keys

# 🔌 How to use

* https://adri567.gitbook.io/autogpt/
